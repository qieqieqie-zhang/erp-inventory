## 📊 系统核心功能模块

### 1. **数据看板** (`/dashboard`)
- 实时业务数据概览
- 关键指标卡片统计
- 销售趋势图表
- 库存预警信息

### 2. **产品管理** (`/product/list`)
- 商品库存管理
- 批量Excel/CSV上传
- 商品信息编辑
- 库存数据导出

### 3. **订单管理** (`/order/list`)
- 亚马逊订单分析（5个时间维度）
- 订单状态跟踪
- 销售数据统计
- 订单报告生成

### 4. **FBA库存管理**
- **FBA库存** (`/fba/inventory`) - 亚马逊FBA中心库存情况
- **FBA预留库存** (`/fba/reserved`) - 预留/待处理库存管理

### 5. **业务报告** (`/business/report`)
- 销售数据分析
- 利润报表
- 业绩趋势
- 自定义报告

### 6. **数据汇总** (`/data/summary`)
- 各模块数据总览
- 跨部门数据整合
- 数据状态时间线

### 7. **系统管理** (管理员专用)
- **用户管理** (`/admin/users`) - 用户账号管理
- **上传日志** (`/admin/upload-logs`) - 文件上传记录

## 📁 数据导入操作

### 支持的文件格式
- Excel文件 (.xlsx, .xls)
- CSV文件 (.csv)

### 导入位置
1. **商品库存数据**：登录后进入「产品管理」页面，点击「上传文件」按钮
2. **订单数据**：登录后进入「订单管理」页面，点击「订单上传」按钮

### 数据验证机制
系统会自动验证SKU字段，无效数据会生成错误报告文件供下载修改。

## 🔧 技术服务支持

### 系统架构
- **后端**: Node.js + Express + MySQL
- **前端**: Vue 3 + Composition API + Pinia
- **UI组件**: Element Plus（除登录页使用原生表单）

### 开发环境
- 端口5173: 前端开发服务器
- 端口3000: 后端API服务器
- 端口3306: MySQL数据库

## 📞 紧急联系

如遇系统故障或紧急问题：
1. 检查所有服务是否运行（3个端口是否正常）
2. 重启服务顺序：
   ```
   1. MySQL数据库 (3306端口)
   2. 后端服务 (d:\codebuddy\库存管理\backend\)
   3. 前端服务 (d:\codebuddy\库存管理\frontend\)
   ```
3. 查看日志文件：
   - `backend/logs/` - 后端错误日志
   - `.workbuddy/memory/` - 系统运行记录
