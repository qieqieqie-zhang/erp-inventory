# 库存日志接入指南

## 概述

所有 SKU 库存的变动都必须记录到 `sku_inventory_logs` 表，通过 `InventoryChangeService` 统一切面自动完成。

## 已集成的模块

| 模块 | 服务初始化 | 写日志位置 |
|------|-----------|-----------|
| 物流上传SKU | `new InventoryChangeService('logistics')` | logistics.controller.js updateSkuList |
| FBA库存上传 | `new InventoryChangeService('fba_inventory')` | fba.inventory.controller.js upload |
| FBA预留上传 | `new InventoryChangeService('fba_reserved')` | fba.reserved.controller.js upload |

## 后续新增功能接入方式

### 1. 在控制器中引入服务

```javascript
const InventoryChangeService = require('../services/InventoryChangeService');
```

### 2. 初始化服务实例

```javascript
// 第二个参数为默认元数据，会贯穿本次请求的所有日志
const logSvc = new InventoryChangeService('logistics', {
  operatorId: req.user ? req.user.id : null,
  operatorName: req.user ? (req.user.realName || req.user.username) : '',
  referenceId: String(logisticsId),
  remarks: '物流记录#xxx上传SKU'
});
```

### 3. 调用日志方法

#### 上传写入（新增）
```javascript
// items 为数组，每个元素需包含 sku_code 和 quantity 字段
await logSvc.logUpload(items, { referenceId: String(recordId) });
```

#### 删除操作
```javascript
await logSvc.logDelete(items, beforeQty, { referenceId: String(recordId) });
```

#### 单独记录一条变动
```javascript
await logSvc.log({
  skuCode: 'SKU123',
  action: 'update',
  beforeQty: 100,
  afterQty: 80,
  meta: {
    operatorId: 1,
    operatorName: '张三',
    referenceId: 'order_456',
    remarks: '手动调整库存'
  }
});
```

#### 批量记录
```javascript
await logSvc.logBatch([
  { skuCode: 'SKU001', action: 'update', beforeQty: 10, afterQty: 20, meta: {} },
  { skuCode: 'SKU002', action: 'delete', beforeQty: 5, afterQty: 0, meta: {} }
]);
```

## 模块标识（module 字段）

| 标识 | 说明 |
|------|------|
| `logistics` | 物流跟踪 |
| `fba_inventory` | FBA库存 |
| `fba_reserved` | FBA预留 |

## 操作类型（action 字段）

| 类型 | 说明 |
|------|------|
| `upload` | 上传/导入 |
| `delete` | 删除 |
| `update` | 更新/调整 |
| `status_change` | 状态变更 |

## 注意事项

- 日志写入失败不应阻断主业务流程，只记录错误
- `operatorId` 和 `operatorName` 必须从 `req.user` 中获取，禁止硬编码
- `referenceId` 用于关联具体业务记录（如物流ID、批次号等）
- 上传类操作 `beforeQty` 固定为 0
- 删除类操作 `afterQty` 固定为 0
